# Changelog

All notable changes to this project will be documented in this file.

## [0.1.1] - 2024-02-18

### Improved
- Added `TaskFactory` for centralized task creation.
- Enhanced docstrings and type hints across all modules.
- Added development dependencies to `pyproject.toml`.
- Expanded test suite with `test_models.py`.

## [0.1.0] - 2023-10-27

### Added
- Initial refactored release.
- Class-based GUI using `customtkinter`.
- Repository pattern for task persistence.
- Service layer for business logic.
- Type hints and docstrings throughout.
- `pyproject.toml` for modern build system.
- Basic unit tests with `pytest`.
- CI workflow with GitHub Actions.
