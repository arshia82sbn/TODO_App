# Changelog

All notable changes to this project will be documented in this file.

## [0.1.1] - 2023-10-28

### Added
- Implemented Factory pattern with `TaskFactory`.
- Implemented Strategy pattern with `BaseRepository` interface.
- Improved type hints and decoupling in `TaskService` and `TaskApp`.

## [0.1.0] - 2023-10-27

### Added
- Initial refactored release.
- Class-based GUI using `customtkinter`.
- Repository pattern for task persistence.
- Service layer for business logic.
- Type hints and docstrings throughout.
- `pyproject.toml` for modern build system.
- Basic unit tests with `pytest`.
- CI workflow with GitHub Actions.
